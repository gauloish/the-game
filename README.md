## Requirements

**Python**: 3.9.1 or Latest

**Pygame**: 2.0.2 or Latest

## Controls

**Select and Move**: a, w, s, d or arrow keys

**Confirm**: Enter

**Shoot**: Space

## Run Game

Run 'the-game.pyw'

Enjoy fun.
